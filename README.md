# FileChronicle
FileChronicle ist ein .NET 10 CLI-Tool für Dateisystem-Snapshots mit intelligenten Vergleichen und Echtzeit-Monitoring. Bietet SHA-256 Hashing, Pattern-Matching, Export nach JSON/CSV/HTML und interaktiven Modus mit Makro-System. Perfekt für Backups, CI/CD und Code-Reviews. Cross-platform für Windows, Linux, macOS. Open Source unter MIT-Lizenz.
